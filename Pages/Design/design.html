Code Break Down

Highlighting My Code Process & Logic

HTML

Key Points:

  • Metadata: The *head* section contains all my metadata. Including my viewport , settings , information about the author and a character set.
  • Styling I have in place the framework for my CSS filing
  • Script I have my script.js where it imports the Javascript module and inialises it with the function initialise (Then the Page It Is On)
  • Structure The page has a nav menu (the old one is commented out due to the updated javascript) , a dynamically generated menu container , the main content section and then my footer.

Code Structure:

  • DOCTYPE html Declares the document type and version of HTML.
  • html lang="en" Defines the document language as English.
  • head: Contains metadata and references to external resources.
  • title Home Page title : Sets the title of the webpage.
  • link rel="stylesheet" href="styles.css">: Links an external CSS file for styling.
  • script type="module">... script : Imports a JavaScript module and initializes it.
  • body Contains the visible content of the webpage.

Javascript

Key Points:

  • Root Path: The root constant stores the base path of the website, which is used as a prefix for all page URLs.
  • Pages Array: The pages array contains objects representing different pages with their names and relative URLs.
  • Navigation Initialization Function: The initialise function dynamically generates a navigation menu based on the provided page data.

Code Structure:

  • The code begins by defining the root constant, which holds the base path of the website.
  • Following that, an array named pages is declared, containing objects representing various pages with their names and URLs.
  • Finally, an initialise function is exported, which dynamically creates a navigation menu based on the provided page information.
  • It is my approach to dynamically generating a navigation menu based on a predefined list of pages.
  • By leveraging the initialise function, I use it to create a set of interactive navigation elements that enhance user navigation within a website.
