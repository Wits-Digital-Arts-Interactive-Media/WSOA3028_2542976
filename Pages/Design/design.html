Design

Font & Color Choices

I'm really into the idea of crafting a super modern website. I'm all about that flow where everything just clicks together – from the vibrant colors playing off each other to the fonts that perfectly complement the vibe. Right now, I'm in that sweet spot of exploration, seeing what works and what doesn't. But I gotta say, I'm liking where I'm at with it all.

Style On Style

I see my website as a serene canvas where every element has its own space. This approach not only enhances the visual appeal but also improves the overall user experience by making navigation intuitive and content easy to consume. By using ample white space, I give my content the spotlight it deserves. This technique helps in emphasizing key messages, calls to action, and important information without overwhelming my visitors. For me, minimalism isn't just about aesthetics; it's about functionality and purpose. Each element on my website serves a clear purpose and contributes to the overall user experience.

Goal Alignment

In my study, I want to figure out how to make websites that do exactly what their owners want them to do. You know when you visit a website and it's easy to find what you need and it looks great? That's what I'm aiming for. By looking at how websites are built and how people use them, I want to help design and make websites that make everyone happy – the owners and the visitors.

Information Structure

For my portfolio website, I wanted a simple and clean structure that would allow visitors to easily navigate and find the content they are looking for. The main sections are the Index/Home page, Portfolio showcasing my projects, an About Me page detailing my background and skills, a section for Essays I've written, Blogposts with regular updates, a Design section, and a Playground area for experimental content. This covers all the key information a visitor might want in an organized manner.

User Flow

The homepage welcomes visitors with a bold intro and provides an overview of what they can expect on the site. From there, they can easily access the different sections through the navigation menu. On the Portfolio page, they can browse through my various projects with visuals and descriptions. The About Me page gives insights into my story, hobbies, technical skills, and tools I work with. The Essays section allows downloading or reading my longform writings. And the Blogposts showcase my latest thoughts and musings. The user can freely explore based on their interests.

    Page Design

    The overall design

    - My goal was to create a clean and minimalistic design that prioritizes content and visual appeal. I believe that simplicity is key to effective communication, and this philosophy guided every design choice I made. By using ample whitespace and a straightforward layout, I ensured that the viewer's attention is immediately drawn to the core messages and visuals.

    - The overall design reflects my unique style and personal brand. It’s a blend of creativity and professionalism, which is crucial for a personal portfolio site. I wanted visitors to not only see my work but also get a sense of who I am as a creator and professional.

    Additional Reaserch:

    WHITE SPACE AND BOLDNESS

    Whitespace acts as a silent guide, creating breathing room between elements to enhance readability and reduce clutter. It fosters a sense of simplicity and elegance, inviting users to focus on key content without feeling overwhelmed. Boldness, on the other hand, serves as a confident statement-maker, drawing attention to important elements such as headlines, buttons, or key messages. When balanced effectively, these elements work in tandem to create a harmonious user experience, where clarity and engagement are seamlessly woven into the fabric of the design.

    Additional Reaserch:

    USER FLOW & FRUSTRATION AVOIDANCE

    User flow is the orchestrated journey that a visitor takes through a website, guiding them from their entry point to their desired destination. It's crucial because it directly influences how users interact with and perceive the site's content and functionality. A well-crafted user flow ensures that navigation is intuitive and seamless, reducing friction and frustration.

    User flow also helps in identifying potential pain points or bottlenecks in the user journey, allowing for iterative improvements that align with user expectations and business objectives. Ultimately, prioritizing user flow in web design promotes a positive experience, encourages engagement, and fosters long-term relationships between users and the website.

    Code Breakdown

    FULL CODE BREAKDOWN Pt.1-

    The structure and layout of my website is built primarily using HTML and CSS. The header contains a navigation menu implemented as an unordered list (ul) with list items (li) representing each menu item. The body section utilizes semantic HTML elements like (header), (nav), (main), and (footer) to organize the content hierarchy. To achieve the grid-like layout for displaying projects and blog posts, I employed CSS Grid or Flexbox, which provide powerful layout capabilities. Additionally, CSS animations and transitions were used to create the subtle particle effect on the landing page.

    FULL CODE BREAKDOWN Pt.2 -

    JavaScript played a crucial role in adding interactivity and dynamic behavior to my website. For instance, the navigation menu may use JavaScript to toggle between the hamburger and expanded states on smaller screens, ensuring a responsive experience. The project sections leverage JavaScript to create interactive image galleries or carousels, enhancing the visual appeal. Additionally, JavaScript Libraries like jQuery were utilized to streamline development and provide reusable components.

    FULL CODE BREAKDOWN Pt.3 -

    To ensure an optimal user experience across devices, I have implemented responsive design techniques using CSS media & queries. These queries allowed me to apply different styles based on screen size, accommodating various viewports. Furthermore, I made use of CSS positive practices to modularize my stylesheets, enabling easier maintenance and code organization. Inline SVG icons or icon fonts were used for scalable, resolution-independent icons throughout the website.

    FULL CODE BREAKDOWN Pt.4 -

    The landing page showcases a minimalistic yet visually appealing design, achieved through careful use of CSS. The particle effect is in there and used to create using the canvas element and JavaScript, leveraging a library with particles.js i coded it myself though. The typography choice, with a clean sans-serif font for headings and body text, contribute to the overall aesthetic. Overall, my website's UI/UX implementation demonstrates a my thoughtful approach to design, layout, interactivity.

    Code Break Down

    HTML

    Key Points:

    • Metadata: The *head* section contains all my metadata, including my viewport settings, information about the author, and a character set.
    • Styling: I have in place the framework for my CSS filing.
    • Script: I have my script.js where it imports the Javascript module and initializes it with the function initialise (then the page it is on).
    • Structure: The page has a nav menu, a dynamically generated menu container, the main content section, and then my footer.

    Javascript - Menu

    Key Points:

    • Root Path: The root constant stores the base path of the website, which is used as a prefix for all page URLs.
    • Pages Array: The pages array contains objects representing different pages with their names and relative URLs.
    • Navigation Initialization Function: The initialise function dynamically generates a navigation menu based on the provided page data.

    Javascript - Typing Animation

    Key Points:

    • Typing Animation: The typeLine function types out a given text character by character, simulating typing. It adjusts the typing speed and removes the caret after typing.
    • Sequential Typing: The startTyping function controls the sequence in which different lines are typed out, ensuring a smooth transition between each word.

    Javascript - Scrolling Animation

    Key Points:

    • addAnimation() Function: This function iterates over elements with the class .scroller, adds a data-animated="true" attribute to each, and duplicates their content within the .scroller__inner element.
    • Cloning Elements: The code clones each item within .scroller__inner, sets aria-hidden to true for accessibility, and appends the cloned item back to the .scroller__inner.
